## 🌟 Overview
CCD Digital Hub is a high-performance, modern web platform designed for **Chai Coffee Darbar**, located near JK Lakshmipat University, Jaipur. The project bridges the gap between campus life and digital convenience, offering a "Kinetic Bento" UI that captures the warmth and energy of the student community.

## ✨ Key Features
* **Kinetic Bento UI**: A non-traditional, asymmetrical grid layout for a "new modern" browsing experience.
* **Real-time Vibe Feed**: Live student reviews and "moods" powered by Supabase Realtime.
* **Interactive Digital Menu**: Comprehensive list of pocket-friendly items (₹15 Chai to ₹249 Peri-Peri Pizza).
* **Heartbeat Micro-interactions**: Sexy, CSS-driven animations that trigger on "favoriting" food items.
* **WhatsApp Order Integration**: Direct "One-Tap" ordering flow to streamline peak-hour traffic.

## 🚀 Getting Started

### Prerequisites
* Node.js 20+
* Supabase Account

### Installation
1.  **Clone the repository**:
    ```bash
    git clone [https://github.com/shivang-portfolio/ccd-digital-hub.git](https://github.com/shivang-portfolio/ccd-digital-hub.git)
    cd ccd-digital-hub
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Set up Environment Variables**:
    Create a `.env.local` file:
    ```env
    NEXT_PUBLIC_SUPABASE_URL=your_url
    NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
    ```

4.  **Run Development Server**:
    ```bash
    npm run dev
    ```
